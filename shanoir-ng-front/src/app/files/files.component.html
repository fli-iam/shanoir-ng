Files

MS Studies

The MS Studies stores study (protocols etc.) and data use agreement files.
View all MS Studies files (JSON)

The MS Studies temporarily writes files into the BIDS tree, /var/bids-data.

MS Datasets

The MS Datasets stores clinical (and preclinical) examination files, called extra-data. For Bruker imports the original Bruker archive is stored as regular extra-data during the import.
View all MS Datasets files (JSON)
The MS Datasets stores BIDS (NIfTI, no DICOM) and EEG files, imported with their corresponding imports.
The MS Datasets stores DICOM files (in dcm4chee-arc, backup-PACS) and processed-datasets files (via manual import or via VIP).

The MS Datasets temporarily writes files into the BIDS tree, /var/bids-data.

MS Preclinical

The MS Preclinical stores PhysiologicalData and BloodGasData files as ExaminationExtraData. PathologyModel files are stored there too.

MS NIfTI Conversion

The MS NIfTI Conversion temporarily generates files in /var/datasets-data.
The MS NIfTI Conversion temporarily writes files into the BIDS tree, /var/bids-data.
