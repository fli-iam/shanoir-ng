1. Select Brainvision archive

Please select a Brainvision or EDF zip archive to import.

The files should be organized like this:

directory_name.zip -> directory_name/dataset_name.vhdr

directory_name.zip -> directory_name/dataset_name.edf

Other files linked to a dataset should have the same name (extension does not matter): dataset_name.other
Multiple datasets can be imported at once (dataset1.edf, dataset2.edf, etc...)
